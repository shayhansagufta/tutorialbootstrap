
=== Typoghrapy ===

Headings

All HTML headings, <h1> sampai <h6>

Bootstrap 4 menggunakan default font-size 16px, dan line-height 1.5
standarnya font-family adalah "helvetica neue", helvetica, arial, 
sans-serif.

selain itu, semua <p> elemen memiliki margin-top: 0 dan margin-bottom: 1rem
(16px secara default)

<h1> - <h6>
bootstrap 4 style HTML headings ( <h1> to <h6> ) dengan font-weight yang
lebih tebal dan font-size yang ditingkatkan.

Display

display digunakan untuk menonjol lebih dari judul normal (ukuran font
lebih besar dan bobot font lebih ringan), dan ada empat kelas untuk
dipilih: .display-1, .display-2, .display-3, .display-4

<small>
di bootstrap 4 <small> elemen HTML digunakan untuk membuat teks sekunder 
yang lebih ringan.

<mark>
bootstrap 4 <mark> elemen HTML dengan warna latar belakang kuning
dan beberapa padding.

<abbr>
<abbr> elemen HTML untuk singkatan dan akronim untuk menampilkan versi 
yang diperluas saat mengarahkan kursor. singkatan memiliki garis bawah
default dan mendapatkan kursor bantuan untuk memberikan konteks tambahan
saat mengarahkan kursor dan kepada pengguna teknologi bantu.

tambahkan .initialism ke singkatan untuk ukuran font yang sedikit lebih
kecil.

<blockquote>
tambahkan .blockquote kelas ke tag <blockquote> saat mengutip blok konten
dari sumber lain
